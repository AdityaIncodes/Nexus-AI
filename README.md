# Nexus-AI

AI SaaS Productivity Platform built with Next.js, TypeScript, Tailwind CSS, Firebase and Gemini API.

## Features

- AI Chat Assistant
- Document Summarization
- Content Generation
- Firebase Authentication
- Chat History
- Responsive Dashboard

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase
- Gemini API
